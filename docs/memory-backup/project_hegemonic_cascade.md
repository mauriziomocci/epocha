Identificato dall'utente il 2026-04-13 come regola fondamentale delle
dinamiche geopolitiche. Pattern storico documentato.

## La cascata

Perdita di controllo delle rotte commerciali critiche
  -> shock economico (prezzi energia/beni, inflazione)
  -> alleati perdono fiducia nella capacita' dell'egemone di proteggerli
  -> alleanze si indeboliscono / si frammentano
  -> capital flight dalla valuta dell'egemone
  -> l'egemone perde capacita' di finanziare la proiezione militare
  -> ulteriore perdita di controllo -> spirale discendente

## Esempi storici

- Suez 1956: UK perde il canale -> crisi della sterlina -> fine egemonia
- Spagna 1600s: perdita dominio navale -> perdita monopolio commerciale
  -> declino economico -> crollo alleanze
- Impero Ottomano: perdita rotte mediterranee -> declino -> perdite
  territoriali

## Implicazione per Epocha

Questa cascata coinvolge 6 fasi simultaneamente: rotte commerciali (11),
economia finanziaria (1c), militare (4), diplomazia (5), energia (9),
psicologia collettiva (10). E' il tipo di feedback cross-sistema che
Epocha deve modellare: ogni fase non funziona in isolamento, l'emergenza
viene dall'interazione.

Il meccanismo deve essere implementato come feedback loop esplicito nel
sistema, non come effetto sperato. Quando le rotte commerciali vengono
interrotte (Fase 11), il sistema deve propagare automaticamente:
trade_disruption -> price_shock (Fase 1c) -> alliance_confidence (Fase 5)
-> capital_flight (Fase 1c) -> military_budget (Fase 4) -> cascata.
